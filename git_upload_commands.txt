# Git Commands to Upload Your Project

# After installing Git for Windows, run these commands in your project folder:

# 1. Initialize Git repository
git init

# 2. Add all files
git add .

# 3. Create initial commit
git commit -m "Initial commit - Procreate Helper Widget"

# 4. Add your GitHub repository (replace YOUR_USERNAME and REPO_NAME)
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# 5. Set main branch
git branch -M main

# 6. Push to GitHub
git push -u origin main

# Example (replace with your actual GitHub username and repository name):
